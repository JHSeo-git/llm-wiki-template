# Agent Skills

This directory is the canonical skill source for this repository.

## Rules

- Always edit skill source files in `.agents/skills/`.
- The Claude Code compatibility path is the symlink `.claude/skills -> ../.agents/skills`.
- To add a new skill, create `.agents/skills/<name>/SKILL.md`.
- Do not place real files or directories under `.claude/skills`.
